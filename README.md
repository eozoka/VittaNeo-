# VittaNeo 🚀

> Plataforma de turismo espacial — Global Solution FIAP 2026

## Descrição

VittaNeo é uma aplicação web de viagens espaciais desenvolvida como projeto da Global Solution. A plataforma permite que usuários explorem destinos orbitais, consultem voos disponíveis e realizem reservas integradas com API REST.

---

## Tecnologias Utilizadas

- **React 18** — Biblioteca de UI
- **Vite** — Build tool e dev server
- **TypeScript** — Tipagem estática
- **Tailwind CSS** — Estilização responsiva
- **React Router DOM v6** — Roteamento SPA
- **Syne + DM Sans** — Tipografia (Google Fonts)

---

## Estrutura de Pastas

- src/
  - assets/ — Imagens e mídias
  - components/ — Componentes reutilizáveis
    - Navbar.tsx
    - Footer.tsx
    - Hero.tsx
    - Destinos.tsx
    - Experiencia.tsx
    - Reservar.tsx
  - pages/ — Páginas da aplicação
    - Home.tsx
    - Voos.tsx
    - Sobre.tsx
    - Contato.tsx
    - Integrantes.tsx
    - FAQ.tsx
  - App.tsx — Roteamento principal
  - main.tsx — Entry point
  - index.css — Estilos globais
